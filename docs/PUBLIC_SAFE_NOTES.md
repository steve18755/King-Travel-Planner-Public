# v6.8 Public-Safe Notes

This build is intended for public GitHub Pages. It removes or masks private family data from the seed/source files.

Excluded or masked:

- Phone numbers
- Email addresses
- Street addresses
- Full birthdays and anniversaries
- KTN / Global Entry / passport values
- Full loyalty and airline account numbers
- Personal profile photos
- Pet photos from inside the home

Do not upload private JSON exports or passport/account images to the public repository. Use the public site as a shell/demo and keep real data in a local import or future secure backend.
