# King Family Travel Planner v6.8 Public-Safe Build

This package is designed for a public GitHub Pages repository. Real family contact data, street addresses, full birthdays, passport/KTN/Global Entry data, full loyalty account numbers, and personal profile/pet photos have been removed or replaced with placeholders. Use local import or a future secure backend for private data.

Upload all contents of this folder to GitHub Pages. Do not commit private JSON exports.
